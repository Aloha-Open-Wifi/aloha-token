# Alohsa Contracts


## Instructions
* Deploy AlohaToken
* Deploy the AlohaSale contract.
* Tranfer the sale tokens to the AlohaSale.
* Call AlohaSale.start with the sale params.

# Setup

Install project deps

```bash
$ npm install
```

# Test

Run tests
```bash
$ npm test
```

# Compile

```bash
$ npm run build
```

You can find the generated abis at ./build/contracts/
